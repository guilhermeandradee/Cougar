Instruções
1. Usando o CMD, abra o diretório 'frontend\' usando o comando cd\ e instale as dependências do projeto com 'npm install'
2. Ainda no prompt de comando, execute o comando 'npm run dev'
3. O CMD irá mostrar uma rota com o título 'local', copie-a e cole-a no espaço para URL's do seu navegador
